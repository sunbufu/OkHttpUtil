# OkHttpUtil
一个OkHttp的工具类库

默认使用OkHttp3.9.0



Add it in your root build.gradle at the end of repositories:

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

Copy
```

Add the dependency

```
	dependencies {
	        compile 'com.github.sunbufu:OkHttpUtil:1.0.0'
	}
```
